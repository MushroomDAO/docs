# 路线图 2026–2027
[English Version](roadmap-en.md)

> **思维链条：** 个体 → 自由与人性 → 菌丝协议协作 → 经济模型探索 → 社区与多样性

我们相信，未来的组织形态是**松散社区与极致个体**。

- **社区最需要什么？** Onboarding + 正反馈激励 + Accountability = **COS72**（社区 OS）。
- **个体需要什么？** 时间、空间、经济可持续，释放自己的想象力、创造力和情感。

## 两个价值坐标系

冷启动与 GToken 价格曲线与这两个坐标系同构——每个阶段完成 = 协议价值跃升。

### 坐标系 A — 问题 × 行动

```
行动等级
    ↑
    │                              ● Phase 5：AAStar 基础设施（持续）
    │                    ●           Phase 4：Mycelium 社区
    │          ●                     Phase 3：CMUBRC 研究
    │    ●                           Phase 2：用爱发电
    ●                                Phase 1：声音建立（完成）
    └────────────────────────────────────→  问题 / 价值维度
```

每解决一个真实问题，驱动一次行动跃升，映射到协议里程碑和融资目标。

### 坐标系 B — 产出 × 成本

```
成本（美元）
    ↑
86,400 │                                  ● Phase 5（AAStar 基础设施）
       │
33,600 │                   ●               Phase 4（社区）
       │
 8,000 │         ●                         Phase 3（研究）
       │
 1,200 │   ●                               Phase 2（基础）
       └────────────────────────────────────→
              产出（论文 / 协议 / SDK / 社区 / DApp）
```

成本随产出非线性增长——随着基础设施被更多用户分摊，单位边际成本持续下降。

**GToken 销售曲线与两个坐标系同构：**
- 每个融资目标 = 坐标系 A 中的一个行动节点
- 获得的融资 = 坐标系 B 中的成本覆盖
- 每阶段 +12% 价格 = 协议价值增长的链上表达
