# Roadmap 2026–2027
[中文版](roadmap-zh.md)

> **Chain of Thought:** Individual → Freedom & Humanity → Mycelium Protocol Collaboration → Economic Model Exploration → Community & Diversity

We believe the future will be defined by a new organizational paradigm: the emergence of **Loose Communities** and **Extreme Individuals**.

- **What does a community need most?** Onboarding + Positive Feedback + Accountability = **COS72** (Community OS).
- **What do individuals need?** Time, space, economic sustainability, and support to release their imagination, creativity, and emotions.

## Two Value Coordinate Systems

The cold launch and GToken price curve are isomorphic to these two systems — every phase funded = a protocol value leap.

### System A — Problem × Action

```
Action Level
    ↑
    │                              ● Phase 5: AAStar Infra (continuous)
    │                    ●           Phase 4: Mycelium Community
    │          ●                     Phase 3: CMUBRC Research
    │    ●                           Phase 2: Powered by Love
    ●                                Phase 1: Voice (done)
    └────────────────────────────────────→  Problem / Value Dimension
```

Each real-world problem drives one action leap. Each leap maps to a protocol milestone and a funding target.

### System B — Output × Cost

```
Cost (USD)
    ↑
86,400 │                                  ● Phase 5 (AAStar Infra)
       │
33,600 │                   ●               Phase 4 (Community)
       │
 8,000 │         ●                         Phase 3 (Research)
       │
 1,200 │   ●                               Phase 2 (Basics)
       └────────────────────────────────────→
              Output (papers / protocol / SDK / community / DApp)
```

Cost grows non-linearly with output — but marginal cost per unit *decreases* as infrastructure is amortized across more users.

**GToken sales curve is isomorphic to both systems:**
- Each funding target = one action node in System A
- Funding received = cost covered in System B
- +12% price per phase = on-chain expression of protocol value increase
