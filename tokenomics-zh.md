# 代币经济学 — GToken & PNTs
[English Version](tokenomics-en.md)

> GToken 不是投资品。它是协议参与凭证——塑造数字公共物品共同体的钥匙。

致敬比特币，社区治理代币 GToken 仅仅发行 2100 万，总量恒定。
GToken 是社区治理 Token，它包含两个核心权利：社区治理权和社区协议参与和使用权。
GToken 拥有者被认为是协议成员，根据协议规定，拥有了对应的治理权和社区协议产品参与和高级功能的使用权。
PNTs 是社区效用代币，作为用户使用社区所有协议产品的基础支付单位，遵守 OpenPNTs 协议，是协议的效用代币。

---

## Token 发行目标

- **正式确立社区所有权**：通过分发 GToken，将协议的所有权和未来发展方向交到用户和社区贡献者手中。
- **实现自我可持续发展**：建立一个由代币持有者管理的社区金库，为协议未来的开发、运营、社区激励等提供资金和激励，摆脱对创始团队的依赖。
- **保护协议的自主与不可篡改性**：通过去中心化治理，确保协议核心特性不被单一实体轻易改变，使其成为一个真正公开、公平的数字未来基础设施。

### 设计逻辑

**80/20 DAO/商业公司：** 商业公司（唯一商业公司/积分联盟）持有 20%，4 年线性解锁。商业公司（20%）+ 核心建设者（12%）= 32%——永远低于 50% 治理门槛，防止中心化控制。

**基础设施挖矿（14%）：** 运行 DVT 验证节点、社区 AI 算力、IPFS 网关的节点按链上可验证贡献获得 GToken。按劳分配，链上追踪。

**冷启动自由兑换（3%）：** 通过无许可链上合约分发，参与早期协议行动即可获得。通过汉威测试——免费分发，无金钱投入，无利润期待。

**守护者空投（3%）：** 四层邀约树。无需收集地址，隐私完全保护。通过社会关系验证防女巫攻击。

### Burn 机制

每次加入和退出协议都会销毁一定数量的 GToken。销毁率由社区多签配置，写入合约。结合 2100 万硬顶，形成自然稀缺性。

```
用户加入协议 → 销毁 X GT
用户退出协议 → 销毁 Y GT
销毁率由社区多签可配置（提案 + 投票）
```

## GToken 权益

GToken **设计上无任何财务回报**。它赋予你：

### 1. 治理权（核心）

| 权利 | 说明 |
|---|---|
| **提案权** | 持有/被委托 ≥1% 总量，可提交 MIP（Mycelium 改进提案） |
| **投票权** | 1 GT = 1 票；特定决策可启用二次方投票 |
| **委托权** | 将投票权委托给活跃社区代表 |
| **协议配置** | 对费用变更、链支持、国库拨款进行投票 |

**治理流程：**
1. **温度检测** —— 治理论坛，约 10,000 GT 非正式投票
2. **共识检测** —— 论坛，需约 30,000 GT
3. **链上提案** —— ≥21 万 GT 提交；7 天投票期；4% 法定人数（84 万 GT）；>50% 通过
4. **时间锁** —— 2 天执行延迟（安全窗口）

### 2. 协议使用权

| 权利 | 说明 |
|---|---|
| **铸造 SBT** | 持有 GToken 铸造跨社区声誉 SBT |
| **注册社区** | 在 Mycelium 部署社区，运营自己的 Paymaster |
| **运营 SuperPaymaster** | 为社区运营 Gas 赞助节点 |
| **任务广场** | 创建和完成赏金任务 |
| **发起协议** | 在生态内启动自己的子协议 |
| **生态优先权** | OpenNest 孵化项目的早期访问资格 |

> **免责声明：** OpenNest 项目的优先访问权不构成投资建议。每个项目独立运营，风险由个人承担。MushroomDAO 对第三方项目表现不承担任何责任。

## PNTs — 实用代币

PNTs 是协议层实用代币，也是基础忠诚积分层。

| 参数 | 值 |
|---|---|
| 初始总量 | 1,000,000,000 |
| 年通胀率 | 2%（每年由治理会议调整） |
| 用途 | 协议内服务支付 · 忠诚积分 · DEX 流动性 |

### 分配

| 类别 | 占比 | 数量 |
|---|---|---|
| 社区国库 | 95% | 950,000,000 |
| Uniswap 流动性池初始化 | 2% | 20,000,000 |
| 创世启动激励 | 3% | 30,000,000 |

治理控制通胀率和国库释放。主要用途：计算网络挖矿奖励、DEX 流动性池、OpenNest 项目激励。

### PNTs 作为积分联盟层

PNTs 是商业积分联盟模型的基础（类比航空里程）：
- 单一发行方（协议）→ 商户是兑换代理商
- 跨商户兑换是 B2B 结算，不是消费者支付
- 用户体验隐形 Web3：扫码 → 赚取/兑换积分 → 无 Gas，无助记词

## xPNTs — 社区实用代币

任何社区都可以通过 `xPNTsFactory` 合约部署自己的 xPNTs 实例。

**示例：aPNTs（AAStar 社区）**
- 以 SuperPaymaster Gas 赞助容量背书
- 价格：$0.018–$0.030 / aPNTs（管理员在范围内调整）
- 无阶梯、无限额、无许可
- 运营商质押 aPNTs 为用户提供 gasless 交易
